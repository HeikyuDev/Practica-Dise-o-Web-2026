# Trabajo Práctico: Fundamentos de HTML y Estructura Semántica

## Ejercicio 1 - Estructura base
Crear un archivo HTML que contenga la estructura básica con las siguientes secciones:
* Etiquetas estructurales principales: `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`
* Configuración del `<head>`: Título del sitio y metadatos (definir un mínimo de 3 etiquetas `<meta>`).

## Ejercicio 2 - Etiquetas estructurales
Dentro de la sección `<main>`, desarrollar los siguientes bloques:
* **Sección principal:** Una sección (`<section>`) que incluya un título `<h2>`, un párrafo `<p>` y una imagen (`<img>` con su respectivo atributo `alt`).
* **Artículo:** Un `<article>` que contenga un título y contenido independiente y propio.
* **Barra lateral:** Un `<aside>` destinado a albergar información complementaria o relacionada de forma secundaria.

## Ejercicio 3 - Listas y tablas
Incorporar los siguientes elementos de organización de datos:
* **Tabla:** Una tabla (`<table>`) con 2 o más filas y columnas, asegurando el uso correcto de encabezados (`<th>`).
* **Lista:** Una lista no ordenada (`<ul>`) que contenga al menos 3 ítems (`<li>`).

## Ejercicio 4 - Enlaces y navegación
Diseñar y construir un menú de navegación funcional dentro de la etiqueta `<nav>` que incluya:
* **Enlaces internos:** Vínculos que redirijan a diferentes secciones dentro del mismo documento HTML (anclas).
* **Enlaces externos:** Vínculos hacia sitios web externos, configurados para abrirse obligatoriamente en una nueva pestaña del navegador.

## Ejercicio 5 - Uso de atributos
Aplicar e implementar de manera práctica los siguientes atributos en diversas etiquetas a lo largo del código:
* **Atributos requeridos:** `id`, `class`, `style`, `lang`, `title`, `align` *(incorporar una aclaración escrita indicando que este último es un atributo obsoleto)*.
* **Documentación:** Explicar claramente el propósito y funcionamiento de cada uno de estos atributos mediante el uso de comentarios internos en el código HTML (``).

## Ejercicio 6 - Sitio ficticio
Desarrollar una mini página web integrada (formato tienda online o blog) que unifique los conceptos anteriores e incluya:
* **Componentes:** Título principal, menú de navegación, maquetación de contenido dividida en secciones, una tabla comparativa de precios, imágenes ilustrativas y un pie de página institucional.
* **Requisitos técnicos:** * Uso de al menos 3 etiquetas semánticas distintas.
  * Uso de al menos 3 atributos globales.
  * Aplicación de estilos visuales embebidos mediante el uso directo del atributo `style`.
